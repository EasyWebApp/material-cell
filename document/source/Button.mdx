Flat buttons and raised buttons are the most commonly used types.

## Buttons

### Flat buttons

Flat buttons are text-only buttons. They may be used in dialogs, toolbars, or inline.
They do not lift, but fill with color on press.

**Outline buttons, i.e. `<Button outline />`, are the equivalent of Material flat buttons.**
**For more details, please refer to [Components/Buttons documentation][1].**

<Example>
    {[
        undefined,
        'Primary',
        'Secondary',
        'Danger',
        'Info',
        'Success',
        'Warning',
        'Dark',
        'Light'
    ].map(color => (
        <Button flat color={color?.toLowerCase()}>
            {color || 'Flat'}
        </Button>
    ))}
</Example>

```TSX
import { render, createCell } from 'web-cell';
import { Button } from 'material-cell/source/Button';

render([
    undefined,
    'Primary',
    'Secondary',
    'Danger',
    'Info',
    'Success',
    'Warning',
    'Dark',
    'Light'
].map(color => (
    <Button flat color={color?.toLowerCase()}>
        {color || 'Flat'}
    </Button>
)));
```

### Raised buttons

Raised buttons are rectangular-shaped buttons. They may be used inline.
They lift and display ink reactions on press.

**Default buttons, i.e. `<Button />`, are the equivalent of Material raised buttons.**
**For more details, please refer to [Components/Buttons documentation][1].**

<Example>
    {[
        undefined,
        'Primary',
        'Secondary',
        'Danger',
        'Info',
        'Success',
        'Warning',
        'Dark',
        'Light'
    ].map(color => (
        <Button className="m-1" color={color?.toLowerCase()}>
            {color || 'Raised'}
        </Button>
    ))}
</Example>

```TSX
import { render, createCell } from 'web-cell';
import { Button } from 'material-cell/source/Button';

render([
    undefined,
    'Primary',
    'Secondary',
    'Danger',
    'Info',
    'Success',
    'Warning',
    'Dark',
    'Light'
].map(color => (
    <Button className="m-1" color={color?.toLowerCase()}>
        {color || 'Raised'}
    </Button>
)));
```

## Floating action buttons

A floating action button represents the primary action in an application, it is used for a promoted action.

<Example>
    <Button float>
        <Icon name="add" />
    </Button>
</Example>

```TSX
import { render, createCell } from 'web-cell';
import { Button } from 'material-cell/source/Button';
import { Icon } from 'material-cell/source/Icon';

render(
    <Button float>
        <Icon name="add" />
    </Button>
);
```
