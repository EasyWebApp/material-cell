## Usage

Chips can be used for various types of entities, including brief information, free form text, or short title. Chips may also contain icons or photos.

<Example>
    <Chip>Chip Label</Chip>
    <Chip icon="C">Chip Label</Chip>
    <Chip image="https://daemonite.github.io/material/apple-touch-icon.png">
        Chip Label
    </Chip>
</Example>

```TSX
import { render, createCell, Fragment } from 'web-cell';
import { Chip } from 'material-cell/source/Chip';

render(
    <>
        <Chip>Chip Label</Chip>
        <Chip icon="C">Chip Label</Chip>
        <Chip image="https://daemonite.github.io/material/apple-touch-icon.png">
            Chip Label
        </Chip>
    </>
);
```

Chips can be deletable. A delete icon can be displayed if a chip is deletable.

<Example>
    <Chip closable>Chip Label</Chip>
</Example>

```TSX
import { render, createCell } from 'web-cell';
import { Chip } from 'material-cell/source/Chip';

render(
    <Chip closable>Chip Label</Chip>
);
```
